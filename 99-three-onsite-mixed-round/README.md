# 99 Three Onsite Mixed Round

## Format
Three unrelated medium questions, like a fast onsite loop.

## Time Target
45-60 minutes total.

## Problem A
Merge all overlapping intervals.

```cpp
vector<vector<int>> mergeIntervals(vector<vector<int>>& intervals);
```

## Problem B
Evaluate an expression in Reverse Polish Notation.

```cpp
int evalRPN(const vector<string>& tokens);
```

## Problem C
Given a binary matrix, return the length of the shortest clear path from top-left to bottom-right using 8-directional movement, or `-1` if impossible.

```cpp
int shortestPathBinaryMatrix(vector<vector<int>>& grid);
```

## Interview Focus
- resetting cleanly between unrelated questions
- choosing the right pattern quickly
- keeping explanations crisp even late in the round
