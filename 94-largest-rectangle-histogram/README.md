# 94 Largest Rectangle In Histogram

## Difficulty
Hard

## Format
Single hard stack problem.

## Time Target
45-60 minutes.

## Prompt
Given bar heights in a histogram, return the area of the largest rectangle.

## Function Signature

```cpp
int largestRectangleArea(const vector<int>& heights);
```

## Interview Focus
- monotonic stack reasoning
- being explicit about what the stack stores
- testing boundaries at the end of the array
