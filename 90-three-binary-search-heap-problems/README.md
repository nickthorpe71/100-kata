# 90 Three Binary Search Heap Problems

## Format
Three medium problems in one round.

## Time Target
45-60 minutes total.

## Problem A
Search for a target in a rotated sorted array.

```cpp
int searchRotated(const vector<int>& nums, int target);
```

## Problem B
Return the `k`th largest element in an unsorted array.

```cpp
int kthLargest(const vector<int>& nums, int k);
```

## Problem C
Given points on a plane, return the `k` closest points to the origin.

```cpp
vector<vector<int>> kClosest(const vector<vector<int>>& points, int k);
```

## Interview Focus
- pattern recognition under time pressure
- choosing heap versus selection versus sorting
- stating complexity tradeoffs clearly
