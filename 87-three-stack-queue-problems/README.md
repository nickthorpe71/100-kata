# 87 Three Stack Queue Problems

## Format
Three small to medium problems in one round.

## Time Target
45-60 minutes total.

## Problem A
Validate whether a string of brackets is balanced.

```cpp
bool isValidBrackets(const string& s);
```

## Problem B
Design a `MinStack` supporting `push`, `pop`, `top`, and `getMin`, all in `O(1)`.

## Problem C
Given an array and window size `k`, return the maximum value in each sliding window.

```cpp
vector<int> maxSlidingWindow(const vector<int>& nums, int k);
```

## Interview Focus
- recognizing monotonic data structures
- keeping API behavior precise
- not overcomplicating straightforward stack problems
