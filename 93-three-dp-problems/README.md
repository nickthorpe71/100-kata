# 93 Three DP Problems

## Format
Three dynamic-programming questions in one round.

## Time Target
45-60 minutes total.

## Problem A
Given a cost array, return the minimum cost to reach the top of the staircase.

```cpp
int minCostClimbingStairs(const vector<int>& cost);
```

## Problem B
Given coin denominations and an amount, return the minimum number of coins needed or `-1` if impossible.

```cpp
int coinChange(const vector<int>& coins, int amount);
```

## Problem C
Return the length of the longest increasing subsequence.

```cpp
int lengthOfLIS(const vector<int>& nums);
```

## Interview Focus
- recognizing 1D DP quickly
- distinguishing state, transition, and base case
- knowing when a greedy instinct is wrong
