# 81 Three Array String Hash

## Format
Three small to medium questions in one round.

## Time Target
45-60 minutes total.

## Problem A
Return the length of the longest substring without repeating characters.

```cpp
int longestUniqueSubstring(const string& s);
```

## Problem B
Group words that are anagrams of each other.

```cpp
vector<vector<string>> groupAnagrams(const vector<string>& words);
```

## Problem C
Return an array where each position contains the product of all other numbers except itself, without using division.

```cpp
vector<int> productExceptSelf(const vector<int>& nums);
```

## Interview Focus
- sliding window fluency
- hash-based grouping
- careful handling of prefix and suffix state
