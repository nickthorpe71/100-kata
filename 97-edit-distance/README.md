# 97 Edit Distance

## Difficulty
Hard

## Format
Single hard DP problem.

## Time Target
45-60 minutes.

## Prompt
Given two strings `word1` and `word2`, return the minimum number of insertions, deletions, and replacements needed to convert `word1` into `word2`.

## Function Signature

```cpp
int editDistance(const string& word1, const string& word2);
```

## Interview Focus
- crisp DP state definition
- base-row and base-column reasoning
- careful walkthrough of a small example
