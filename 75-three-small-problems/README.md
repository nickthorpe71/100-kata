# 75 Three Small Problems

## Format
Three small unrelated questions in one round.

## Time Target
45-60 minutes total. Aim for 12-15 minutes per problem plus review.

## Problem A
Given `nums` and `target`, return the two indices whose values sum to `target`.

```cpp
pair<int, int> twoSumIndices(const vector<int>& nums, int target);
```

## Problem B
Given a string `s`, return `true` if it can become a palindrome after deleting at most one character.

```cpp
bool validNearPalindrome(const string& s);
```

## Problem C
Given daily temperatures, return for each day how many days you have to wait until a warmer temperature. If none exists, return `0`.

```cpp
vector<int> daysUntilWarmer(const vector<int>& temperatures);
```

## Interview Focus
- switching contexts without losing structure
- choosing the right data structure quickly
- stating complexity before moving on
