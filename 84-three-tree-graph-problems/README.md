# 84 Three Tree Graph Problems

## Format
Three related data-structure questions in one round.

## Time Target
45-60 minutes total.

## Problem A
Return the maximum depth of a binary tree.

```cpp
int maxDepth(TreeNode* root);
```

## Problem B
Given a BST, return its `k`th smallest value.

```cpp
int kthSmallest(TreeNode* root, int k);
```

## Problem C
Given a grid of `"1"` and `"0"`, return the number of islands.

```cpp
int countIslands(vector<vector<char>>& grid);
```

## Interview Focus
- recursion state definition
- DFS versus iterative traversal tradeoffs
- consistent edge-case handling
